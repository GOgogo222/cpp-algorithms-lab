# 算法笔记

这里存放算法学习和刷题过程中的文字笔记，用来补充 `leetcode/` 中的代码题解。

## 目录

- `algorithm-practice/`：数据结构、算法基础和日常刷题记录。
- `algorithm-practice/leetcode/`：按题号保存的 LeetCode 题目笔记。
- `competitive-programming/`：天梯赛、竞赛训练和阶段复盘。
- `cpp-stl.md`：C++ STL 常用容器、算法和写法记录。

## 记录方式

- 一题一文件，优先保留题号、题名、关键思路和易错点。
- 阶段性训练放入对应专题目录，避免散落在 Obsidian 中。
