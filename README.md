# C++ 算法实验室

这个仓库集中存放 C++ 课程作业、基础练习、LeetCode 题解代码，以及从 Obsidian 迁出的算法刷题笔记。

## 仓库结构

- `assignments/`：C/C++ 课程作业和基础练习。
- `leetcode/`：按题号归档的 LeetCode 题解代码。
- `notes/algorithm-practice/`：数据结构、算法基础和 LeetCode 刷题笔记。
- `notes/competitive-programming/`：天梯赛、竞赛训练和比赛复盘。
- `notes/cpp-stl.md`：C++ STL 学习笔记。

## 维护约定

- 代码题解放在 `leetcode/`，按题号和题名归档。
- 思路、错题、复盘放在 `notes/`，作为长期可查的学习记录。
- Obsidian 知识库只保留入口链接，不再维护算法刷题原文。
